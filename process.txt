ResearchCompiler

Receives an input -> Search Term

Using Search Term, it will find all relevant information on it in the web

It will collect:
1. Definition
2. Related Articles / Links
3. Terms and Related Subjects

How to collect:
Dictionary Search -> Get Link
Wikipedia Search -> Get Link
Google Search -> Get top 10 links
Google Scholars Search -> Get top 10 links
Google News -> 2 pages worth of links

Outputs all this into an Excel Sheet
